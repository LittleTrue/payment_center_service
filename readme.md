### 关于支付组件
基于service-client结构下IOC组件, 基于官方DEMO的规则面向过程使用guzzle简单实现接口通讯。

### 支持的业务场景
* 1、普通和国际支付
* 2、自助清关报关

##### 目前对接的支付接口功能和对应的:
* 1、普通微信支付 V2版本 (JSAPI、小程序、付款码支付、Native支付、通用查询、退款)
* 2、国际微信支付 V3版本 (Native支付、通用查询、退款)
* 3、普通支付宝支付 (付款码支付、通用查询、退款)
* 4、国际支付宝支付 (扫码支付、通用查询、退款)
* 5、微信报关 V2版本 (报关、报关查询、重推)
* 6、支付宝报关 (报关、报关查询、重推)
